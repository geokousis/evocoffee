# EvoCoffee

Terminal-inspired inventory tracker for EvoLab coffee supplies.

## Run locally
Open `index.html` in a browser.

## Push to GitHub
```bash
git init
git add index.html styles.css app.js README.md .gitignore DEPLOYMENT.md
git commit -m "Initial EvoCoffee site"

git branch -M main
git remote add origin <YOUR_GITHUB_REPO_URL>
git push -u origin main
```

## Notes
All data is stored in the browser (localStorage). Export/Import is available in the UI.
